About the app

This app allows for the visualisation and exploration of data obtained from a BICS survey about businesses' responses to the pandemic.

The app uses d3.js for the visualisation of the graphs, and for hosting the website, Github Pages was used.

To query the RDF data, a server loaded the TTL files and they were queried with SPARQL which was then converted into javascript objects to be used with the d3.js library for the frontend.

Multidimensional Data Visualisation of the graphs

Filtering: Above the graphs, users may switch between different filters for the visualisation and alternate between different filters.

Selections: By hovering over the bar chart, the user may select/filter the pie chart data based on the industry. Similarly, by hovering over the pie chart, the user may select/filter the bar chart data based on the workforce size.

The use of both bar charts and pie charts, allows for multi-view and exploration of the data.

Keys and legends are used for the user to understand the data in the charts.

Graph Choice

The bar chart was chosen because it allows for good visualisation of frequency distribution, summarising the data in an interpretable manner.

The combination of bar charts and pie charts allows for multi-view and exploration of the data, with the pie charts allowing for good visualisation of the proportion of the data.
